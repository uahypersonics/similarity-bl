# Use Or Export Profiles

After solving, profiles can be used directly in Python or MATLAB, written to
files, or compared against reference data.

Typical uses include:

- comparing against CFD or experimental boundary-layer profiles
- generating locally self-similar baseflow profiles for stability analysis
- sweeping stations when edge conditions vary along a body
- exporting profiles for MATLAB, Tecplot, or other post-processing tools

The validation cases under [Validation and Verification](../vnv/index.md) show
complete reproducible examples.

For runnable examples, see [CLI Usage](../user_guide/cli_usage.md),
[API Usage](../user_guide/api_usage.md), and
[MATLAB Usage](../user_guide/matlab_usage.md).
